12月 29 2013

在windows下习惯了chrome，虚拟机里装了centos6.5。
网上找到了资料：

1.在/etc/yum.repos.d/google-chrome.repo  
2.加入：  
  [google-chrome]
  name=google-chrome
  baseurl=http://dl.google.com/linux/chrome/rpm/stable/$basearch
  enable=1  
  gpgcheck=1  
  gpgkey=https://dl-ssl.google.com/linux/linux_signing_key.pub  
3.yum install google-chrome-stable

照着做，结果出现libstdc++.so.6(GLIBCXX_3.4.15)(64bit) ...

网上查到原因：chrome linux开发人员说：“Centos/RHEL6 are unfortunately too old”。以上方法失败，继续查找:
一条命令:
yum install http://people.centos.org/hughesjr/chromium/6/x86_64/RPMS/chromium-31.0.1650.63-1.e16.x86_64.rpm
安装成功！
关于版本选择：可以到http://people.centos.org/hughesjr/chromium/6/x86_64/RPMS/ 下选择相应的安装包
目前有28版本和31版本

删除命令：yum remove chromium.x86_64
